# **App Name**: MediStore

## Core Features:

- New Bill Creation: Enable the creation of new bills with patient and doctor information.
- View Bills: Display a comprehensive list of previous bills with a search function for patient lookup.
- Add Stock: Add new medication stock with details like name, company, type, batch, and expiry date.
- View Stock: Show available stock with color-coded indicators for stock levels and expiry dates.
- View Reports: Generate reports for billing and stock, filtered by date range.
- Manage Staff: Use sign up form followed by a qr code from admin profile to enable the staff.
- Settings: Provide settings for printer type, invoice title, staff permissions, and local bill storage.

## Style Guidelines:

- Primary color: A calming light blue (#64B5F6) to evoke trust and health.
- Background color: Very light blue (#E3F2FD) to maintain a clean and professional look.
- Accent color: A contrasting orange (#FFB74D) to highlight key actions and information.
- Clean and professional font.
- Use clear and recognizable icons for navigation and actions.
- Organize the dashboard with clear sections for each function, ensuring easy navigation.
